# Real-Life Example

Socho tum ek school ke principal ho aur tumhare paas **1000 students ka data** hai.

Data mein hai:

| Name  | Age | Marks | City    |
| ----- | --- | ----- | ------- |
| Ali   | 20  | 85    | Lahore  |
| Sara  | 21  | 92    | Karachi |
| Ahmed | 19  | NaN   | Lahore  |

Ab principal tumse poochta hai:

* Average marks kitne hain?
* Kitne students Lahore se hain?
* Kis student ke marks missing hain?
* Sabse zyada marks kis ke hain?
* Kya data mein koi problem hai?

In sawalon ke jawab dene ka process hi **EDA** hai.

---

# EDA ka Matlab

**Exploratory** = Explore (data ko dekhna)

**Data** = Dataset

**Analysis** = Analyze karna

**EDA = Data ko samajhna.**

---

# Machine Learning se pehle EDA kyu?

Suppose tumhare paas ye data hai.

```text
Age

21
22

```

Kya koi insan ki age **200** ho sakti hai?

❌ Nahi.

EDA ke through tum ye problem pakad loge.

---

Ek aur example:

```text
Name     Salary
Ali      50000
Sara     NaN
Ahmed    60000
```

Yahan salary missing hai.

EDA batata hai:

> "Bhai, pehle data saaf karo (clean karo), phir model banao."

---

# EDA ka Main Goal

Dataset ko samajhna.

Questions:

* Kitni rows hain?
* Kitne columns hain?
* Missing values kitni hain?
* Average kya hai?
* Maximum kya hai?
* Minimum kya hai?
* Duplicate rows hain?
* Outliers hain?
* Data balanced hai?

---

# EDA ka Step-by-Step Workflow

## Step 1: Load Data

```python
import pandas as pd

df = pd.read_csv("netflix_titles.csv")
```

Dataset memory mein aa gaya.

---

## Step 2: First 5 Rows Dekho

```python
df.head()
```

Output

```text
show_id
type
title
director
country
```

Purpose:

"Dekho data kis type ka hai."

---

## Step 3: Last Rows

```python
df.tail()
```

Purpose:

End ka data bhi dekh lo.

---

## Step 4: Shape

```python
df.shape
```

Output

```text
(8807, 12)
```

Matlab

* 8807 rows
* 12 columns

---

## Step 5: Columns

```python
df.columns
```

Output

```text
Index(['show_id',
'type',
'title',
'director',
'country',
'release_year'])
```

Purpose:

Columns ke naam dekhna.

---

## Step 6: Data Types

```python
df.info()
```

Output

```text
show_id object
release_year int64
rating object
```

Purpose:

Kaunsa column number hai aur kaunsa text.

---

## Step 7: Missing Values

```python
df.isnull().sum()
```

Output

```text
director      2634
country        831
rating           4
```

Matlab

Director column mein 2634 missing values hain.

---

## Step 8: Descriptive Statistics

```python
df.describe()
```

Output

```text
count
mean
std
min
max
```

Purpose

Numeric columns ko summarize karna.

---

## Step 9: Unique Values

```python
df['type'].unique()
```

Output

```text
Movie
TV Show
```

---

## Step 10: Value Counts

```python
df['type'].value_counts()
```

Output

```text
Movie      6131
TV Show    2676
```

Purpose

Har category kitni baar aayi.

---

## Step 11: Duplicate Rows

```python
df.duplicated().sum()
```

Purpose

Kya same row dobara aa gayi?

---

## Step 12: Remove Missing Values

```python
df.dropna()
```

Ya

```python
df.fillna(0)
```

---

## Step 13: Visualization

Ab graphs bante hain.

```python
import seaborn as sns

sns.countplot(data=df, x='type')
```

Graph bata dega

Movie ███████████████

TV Show █████

# Netflix Dataset Example

Suppose tumhare manager ne bola:

> "Netflix ka data analyze karo."

Tum seedha model nahi banaoge.

Tum ye steps follow karoge:

```python
import pandas as pd

df = pd.read_csv("netflix_titles.csv")

# First look
df.head()

# Shape
df.shape

# Columns
df.columns

# Data types
df.info()

# Missing values
df.isnull().sum()

# Statistics
df.describe()

# Movies vs TV Shows
df['type'].value_counts()

# Remove missing values
df.dropna()
```

Ye pura process **EDA** hai.

---

# AI/Data Science Interview Definition

**Q: What is Exploratory Data Analysis (EDA)?**

**Answer:**

> Exploratory Data Analysis (EDA) is the process of understanding, summarizing, cleaning, and visualizing a dataset before building machine learning models. It helps identify missing values, duplicates, outliers, data types, distributions, and relationships between variables so that the data is ready for analysis and modeling.

## 💡 Golden Rule (Jo har Data Scientist follow karta hai)

```
Collect Data
      ↓
EDA (Understand + Clean + Visualize)
      ↓
Feature Engineering
      ↓
Train Machine Learning Model
      ↓
Evaluate Model
```
